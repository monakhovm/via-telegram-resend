# via-telegram-resend

## USAGE

### Create/edit ```.env``` file - set enivironment variables:
```
TELEGRAM_BOT_TOKEN=<bot_token>
MAIN_TELEGRAM_CHANNEL=<ID_of_main_channel>
```

### Create/edit ```channels``` file - fill it with channelto resend ID`s:
Example:
```
123456789
-100987654321
-987654321
```

### Run application
